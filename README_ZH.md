###  **盒子IM 桌面端


1. 盒子IM是一个仿微信实现的网页版聊天软件，不依赖任何第三方收费组件。
2. 本项目是盒子IM桌面端，基于盒子IM web端迁移至electron框架。
3. 开发使用了 electron-vite-template(https://github.com/umbrella22/electron-vite-template) 作为electron开发基础模板
4. 项目模仿QQ最新版页面布局风格

#### 本地启动
1.安装运行环境
- node:v18.19.0

2.启动

npm install
npm run dev
